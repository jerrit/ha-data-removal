# Data Removal

Monitor 35 data broker sites for your personal information and automate opt-out
removal requests — all from your Home Assistant dashboard.

All data stays local. Nothing is sent externally.

## Features

- Scans Whitepages, Spokeo, BeenVerified, Radaris, and 31 more
- Automated form-filling for opt-out requests
- Generates email templates for email-based removals
- Multi-user support — monitor multiple family members
- Auto-rescans sites 30 days after removal requests
- Live progress bar during scans

## Setup

Configure your personal details in the addon **Configuration** tab, then open
the Web UI to start scanning.
