# Data Removal

Monitor 35+ data broker websites for your personal information and automate
opt-out removal requests — all from a dashboard inside Home Assistant.

All data stays local on your machine. Nothing is sent to external services.

## Features

- Scans Whitepages, Spokeo, BeenVerified, Radaris, MyLife, and 30+ more
- Automated form-filling for opt-out requests
- Email template generator for email-based removals
- Multi-user support — monitor multiple family members
- Auto-rescans sites 30 days after removal requests
- Live progress bar during scans
- Add custom data broker sites
- Optional password protection for the web UI

## Setup

1. Start the add-on and open the **Web UI**
2. Click **Add a profile** and enter your personal details
3. Click **Scan All Sites** to begin your first scan

Personal information is managed entirely through the web UI.
See the [full documentation](DOCS.md) for details.
