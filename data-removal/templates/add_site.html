Tips for finding search URL templates

  • Go to the site and search for a common first/last name manually
  • Copy the URL from the results page and replace your name and state with {first}, {last}, {state}
  • Example: https://site.com/people/john-smith/il → https://site.com/people/{first_last}/{state}
  • If the site uses a form (not URL-based search), leave the search template blank — the scanner will still try to visit the homepage and check for your name
